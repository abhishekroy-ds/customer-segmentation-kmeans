# Mall Customer Segmentation using K-Means Clustering

## Project Overview
This project uses K-Means clustering to segment mall customers based on their annual income and spending score.

## Dataset
Dataset used: Mall Customers Dataset

Features:
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## Objective
To group customers into different segments for targeted marketing strategies.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis
- Feature Selection
- Finding optimal K using Elbow Method
- Applying K-Means Clustering
- Visualization of clusters

## Results
Identified 5 customer segments
High-income high-spending customers
Low-income low-spending customers
Potential target groups for marketing campaigns
